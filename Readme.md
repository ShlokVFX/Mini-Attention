# Mini-Attention

> Building Attention Mechanisms from Research Papers — From PyTorch to Kernels

Mini-Attention is a research-driven educational repository focused on implementing modern attention mechanisms from scratch across multiple system layers:

- PyTorch (baseline reference)
- Python DSLs (Triton / Helion)
- CUDA kernels (performance implementations)

The goal is to bridge the gap between:

Research Papers → Mathematical Understanding → Systems Kernels → GPU Optimization
